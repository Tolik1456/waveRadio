Одностраничный сайт "Радио - W-wave"
Тема проекта:
Frontend - часть сайта Радио компании.

В данном проекте реализованы:

- Pixel Perfect
- БЭМ - наименование классов
- Слайдеры, аккордеоны
- Hover, active эффекты
- Flexbox - верстка без использования фреймворков
